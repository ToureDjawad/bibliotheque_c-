INSTRUCTIONS DE COMPILATION

Le projet utilise un Makefile pour automatiser la compilation.

PREREQUIS :

- Un compilateur g++
- Un terminal (Linux, Mac ou Windows )

COMMANDES A TAPER DANS LE TERMINAL :
- Pour nettoyer le projet  : make clean
- Pour compiler le projet : make

LANCEMENT :
Une fois compilé, lancez l'application avec la commande : ./app

FICHIERS IMPORTANTS :
- app : L'exécutable du programme.
- library.db : Fichier de sauvegarde de la bibliothèque.
- app.conf : Fichier de configuration
- Makefile : Script de compilation.
- README.txt : Ce fichier d'aide.
